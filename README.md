# All Searching

**목표** : Kakao Open API 이용한 검색 어플리케이션 개발 


**개발언어** : Kotlin </br>
**IDE** : Android Studio </br>
**네트워킹** : Retrofit 2.0, OkHttp </br>
**아키텍처** : MVVM, Clean Architecture </br>
**DI** : Hilt </br>
**Database** : Room </br>
**그 외** : ViewBinding, DataBinding, Coroutine, Android Jetpack, WebView </br>
